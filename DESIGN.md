# Design

## Theme
- Font family: `Instrument Sans`, sans-serif.
- Color Strategy: Restrained (neutral zinc tones + single accent color based on theme).

## Colors
- **Light Theme**:
  - Background: Zinc 50 (`#fafafa`)
  - Accent: Neutral 800 (`#262626`)
  - Border: Gray 200 (`#e5e5e5`)
- **Dark Theme**:
  - Background: Zinc 950 (`#0a0a0a`)
  - Accent: White (`#ffffff`)
  - Border: Zinc 800 (`#262626`)

## Components
- **Buttons**: Flux UI variants (primary, secondary, danger, ghost).
- **Forms**: Outlined controls with accent ring indicator on focus.
- **Table**: Flux UI tables with border separators and clean padding.
- **Card**: Clean boundaries, minimal drop shadows.
